# 文件保险箱 (FileStrongbox)

一款简洁易用的文件加密工具，采用拖放操作，安全可靠。

## 功能特色

### 简单直观的操作
- **拖放加密** - 将文件或文件夹拖入加密区即可加密
- **拖放解密** - 将加密文件或文件夹拖入解密区即可解密
- **批量处理** - 支持同时处理多个文件和整个文件夹

### 安全可靠
- **AES-256-GCM** - 采用军事级加密算法，确保数据安全
- **PBKDF2 密钥派生** - 10万次迭代，有效防止暴力破解
- **完整性验证** - GCM 模式自带认证，防止数据被篡改
- **安全文件处理** - 先加密后删除，确保数据不丢失

### 灵活的文件名选项
- **完全重命名** - 文件名加密为哈希值，完全隐藏原始文件名
- **保留原名** - 保持原始文件名不变
- **新后缀** - 文件名加密 + 自定义后缀（如 .data）

## 使用方法

### 加密文件
1. 将文件或文件夹拖入左侧「加密区」
2. 输入密码并确认
3. 等待加密完成

### 解密文件
1. 将加密文件拖入右侧「解密区」
2. 输入正确的密码
3. 等待解密完成
